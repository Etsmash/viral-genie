# 🧞‍♂️ Viral Genie

Viral Genie is a futuristic, AI-powered guessing game focused on viral internet products and trends from 2024 to 2026. 

## 🚀 Features
- **Dynamic Trend Search**: Fetches the latest internet trends from `trends.json` on load.
- **Intelligent Guessing**: Powered by a custom entropy-based scoring engine.
- **Premium UI**: Modern dark mode with Glassmorphism and vibrant animations.
- **Auto-Deployment**: Hosted on GitHub Pages via GitHub Actions.

## 🕹️ Live Demo
[Viral Genie Live](https://Etsmash.github.io/viral-genie)

## 🛠️ How to Update
To add new viral products to the game without touching the code, simply edit the `trends.json` file in this repository.

## 📂 Project Structure
- `index.html`: The main entry point.
- `game.js`: Core engine and dynamic fetching logic.
- `products.js`: Static database of viral products.
- `trends.json`: Dynamic database for live updates.
- `style.css`: Premium layout and design.
